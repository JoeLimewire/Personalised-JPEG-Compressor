What is JPEG Compression?

JPEG stands for Joint Photographic Experts Group.
It is a standard for compressing images all across the world and it does this because it can reduce the file size of images massively while maintaining the image quality. Why don't you try to reduce the image quality until you can see differences in the images?
Prettly low right?
It does this by splitting it into 8x8 block, and analysing its spacial frequancies.

All this means, it grabs 8x8 blocks of pixels and it trys to find certain shapes to recreate the block. Then all it has to store is what shapes it uses then it can make the block again.
